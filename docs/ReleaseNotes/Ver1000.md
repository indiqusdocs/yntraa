# Introduction
We are excited to announce the general availability of Yntraa Cloud Platform, a modern cloud infrastructure platform designed to support enterprise applications, data platforms, and digital services.

This first release introduces a comprehensive set of foundational cloud services that enable organizations to provision, manage, and scale infrastructure resources on demand. The platform combines networking, compute, storage, database, cloud assure, security services, and other services into a unified environment, allowing teams to deploy workloads with greater flexibility, performance, and control.

## Services Available in this Release

### Networking Services
Build secure and isolated network environments for cloud workloads.

Key capabilities:
- Virtual private networks (VPCs)
- Subnet configuration and IP management
- Load balancing for application traffic
- Secure connectivity between cloud services

### Compute Services
Provision and manage virtual compute resources to run applications and services.

Key capabilities:
- Virtual machine provisioning
- Flexible CPU and memory configurations
- Rapid instance deployment
- Lifecycle management for compute resources
- Managed Compute

### Database Services
Deploy and manage database systems with simplified provisioning and management.

Key capabilities:
- Managed database deployment
- Automated configuration and provisioning
- Backup and recovery support
- Database performance monitoring

### Storage Services
Store and manage data using scalable and highly available storage services.

Key capabilities:
- Block Volumes for compute workloads
- Object storage for unstructured data
- Persistent storage volumes
- Scalable storage capacity

### Other Services
Protect applications, infrastructure, and data with built-in security capabilities.

Key capabilities:
- HSM and KMS
- UTM
- WAF
- SIEM
- Load testing
- Security monitoring and policy enforcement
