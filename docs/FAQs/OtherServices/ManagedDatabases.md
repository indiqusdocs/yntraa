# Managed Databases


<details><summary>**What is Yntraa Cloud Managed Database as a Service (MDBaaS)?**</summary>Yntraa Cloud MDBaaS is a fully managed database solution that offers enterprises scalable, secure, and high-performance databases without the need to manage the underlying infrastructure.</details>

<details><summary>**Which databases are supported by Yntraa Cloud MDBaaS?**</summary>Yntraa Cloud MDBaaS supports various database engines like MySQL, PostgreSQL, MSSQL, MariaDB, MongoDB, Redis, Cassandra, Opensearch, Elasticsearch, Hadoop, ScyllaDB, Couchbase.</details>

<details><summary>**What are the key benefits of using Yntraa Cloud MDBaaS?**</summary>Key benefits include Fully managed service, high availability, scaleability, enhanced security, and 24/7 support.</details>

<details><summary>**How does Yntraa Cloud MDBaaS differ from traditional database management?**</summary>Unlike traditional databases, Yntraa Cloud MDBaaS handles the infrastructure, backups, patching, and scaling automatically, allowing businesses to focus on application development and data management.</details>

<details><summary>**What does a typical Cloud Assessment cover?**</summary>It may include infrastructure analysis, workload assessment, cost analysis, compliance review, cloud readiness scoring, network mapping, and risk identification.</details>

<details><summary>**Can I migrate my existing databases to Yntraa Cloud MDBaaS?**</summary>Yes, Yntraa Cloud provides tools and support for seamless migration of existing
databases to the managed service.</details>

<details><summary>**What pricing plans are available for Yntraa Cloud MDBaaS?**</summary>Yntraa Cloud offers flexible pricing models based on resource usage, including pay-as-you-go, with monthly billing.</details>

<details><summary>**How is the price calculated?**</summary>Pricing is based on the database engine, instance type, storage, data transfer, support, license and additional features like backups and security.</details>

<details><summary>**Are there any hidden fees or charges?**</summary>No, Yntraa Cloud provides transparent pricing with a detailed breakdown of costs. Any additional features or services are clearly outlined before purchase.</details>

<details><summary>**Do you offer a free trial for Yntraa Cloud MDBaaS?**</summary>Yes, a free trial is available, allowing customers to experience the service before committing to a paid plan.</details>

<details><summary>**Can I change my plan after signing up?**</summary>Yes, customers can upgrade or downgrade their plans at any time based on their changing needs.</details>

<details><summary>**How does Yntraa Cloud MDBaaS ensure high availability and disaster recovery?**</summary>Yntraa Cloud MDBaaS uses multi-zone deployments, automated backups, and replication to ensure high availability and disaster recovery.</details>

<details><summary>**What security measures are in place for Yntraa Cloud MDBaaS?**</summary>Security measures include encryption, regular security patches, and compliance with industry standards.</details>

<details><summary>**Can I scale my database instances as my business grows?**</summary>Yes, Yntraa Cloud MDBaaS allows you to scale your database instances horizontally (adding more instances).</details>

<details><summary>**How do automated backups work?**</summary>Automated backups are performed regularly and can be customized based on your requirements. These backups are stored securely and can be used for point-in-time recovery.</details>

<details><summary>**What kind of monitoring and logging features are available?**</summary>Yntraa Cloud MDBaaS offers real-time monitoring, detailed logs, and alerts to
help you keep track of your database performance and security.</details>

<details><summary>**How can I get started with Yntraa Cloud MDBaaS?**</summary>You can get started by signing up on the Yntraa Cloud portal, selecting your preferred database engine, and configuring your instance through an easy-to-use interface.</details>

<details><summary>**What support options are available for Yntraa Cloud MDBaaS users?**</summary>Yntraa Cloud provides 24/7 customer support through various channels, including live chat, email, and phone support. Additionally, a dedicated account manager is available for enterprise customers.</details>

<details><summary>**Can I customize my database configuration?**</summary>Yes, you can customise various aspects of your database, including instance size, storage type, security settings, and backup schedules.</details>

<details><summary>**What happens if I need to cancel my service?**</summary>You can cancel your service at any time. However, depending on your pricing plan, certain terms may apply regarding refunds or data retention.</details>

<details><summary>**How is Yntraa Cloud MDBaaS integrated with other Yntraa Cloud cloud services?**</summary>Yntraa Cloud MDBaaS integrates seamlessly with other Yntraa Cloud cloud
services, such as Yntraa Cloud FlexiDB and Yntraa Cloud SutraDB, offering a comprehensive cloud ecosystem.</details>

<details><summary>**Are there any special compliance certifications that Yntraa Cloud MDBaaS holds?**</summary>Yntraa Cloud MDBaaS complies with various industry standards, including ISO
27001, GDPR, HIPAA, and others, ensuring that your data is managed in a secure and compliant manner.</details>

<details><summary>**What is the typical deployment time for a new database instance?**</summary>Deployment time for a new database instance is typically a few minutes,
depending on the complexity of your configuration.</details>

<details><summary>**Can I manage multiple database instances through a single Yntraa Cloud
account?**</summary>Yes, Yntraa Cloud provides a unified dashboard where you can manage multiple database instances, monitor performance, and access logs.</details>

<details><summary>**Does Yntraa Cloud offer training or onboarding sessions for new users?**</summary>Yes, Yntraa Cloud offers training sessions, webinars, and documentation to help new users get up to speed with the platform.</details>
