# End Point Security

Endpoint Security protects end-user devices such as desktops, laptops, and mobile phones from malicious and unauthorized activities through advanced security practices and technologies. It safeguards corporate networks by securing the devices employees use to access applications, systems, and data, preventing unauthorised access, data breaches, and cyber threats across the organisation.

To activate the desired Endpoint Security service, perform the following steps:
1. Navigate to **MANAGED SECURITY** > **EndPoint Security**![endpointsecurity](img/endpointsecurity.png)
2. Click the **ACTIVATE** button.![endpointsecurityactivation](img/endpointsecurityactivation.png)
   3. Select the I have read and agreed to the **End User License Agreement** and **Privacy Policy** option, and click **CONFIRM ACTIVATION** button.
