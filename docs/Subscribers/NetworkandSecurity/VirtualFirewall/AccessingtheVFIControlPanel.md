# Accessing the VFI Control Panel

The Virtual Firewall Service is delivered as an integration using OS images pre-loaded with Yntraa Software Firewall.

Navigate to **Network and Security > Virtual Firewalls**, and click the **Overview** tab. The following screen appears:

![vffirewallprovider](img/vffirewallprovider.png)

To access the Virtual Firewall Control Panel, click the **Launch Console** button .
