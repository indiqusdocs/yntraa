# Introduction
Yntraa Cloud Managed Database as a Service (MDBaaS) is a fully managed database offering that enables you to **deploy, operate, and scale databases on Yntraa Cloud without managing the underlying infrastructure or database operations**.

With MDBaaS, Yntraa assumes responsibility for:
- Database provisioning and deployment
- High availability configuration (where applicable)
- Day-to-day operational management
- Patching, monitoring, and incident handling as per the subscribed plan

You interact with the service through a **simplified QuickPlan-based provisioning workflow**, allowing rapid activation of databases while maintaining enterprise-grade governance and control.

MDBaaS is designed for:
- Application teams that want faster time to market
- Enterprises seeking operational consistency
- Customers who prefer outcome-based database services rather than infrastructure management.

## Purpose and Scope 

This document serves as the official customer user guide for Yntraa Cloud – Managed Database as a Service (MDBaaS).

**The purpose of this guide is to:**

- Explain how customers can request and activate managed database services on Yntraa Cloud
- Describe the end-to-end provisioning workflow using the QuickPlan model
- Clarify customer inputs, system actions, and operational handoffs
- Set clear expectations on what is managed by Yntraa and what is controlled by the customer

**Yntraa is responsible for:**

- Database software installation and configuration
- Infrastructure provisioning (compute, storage, network)
- Availability setup as per selected plan
- Operational monitoring and maintenance

**Customers are responsible for:**

- Providing accurate input parameters during provisioning
- Application integration and data usage
- Database-level access management for application users

| **Layer**                               | **Responsibility** |
| --------------------------------------- | ------------------ |
| Physical Infrastructure                 | Yntraa             |
| Virtualization and Compute              | Yntraa             |
| OS and Patching                         | Yntraa             |
| Database Installation and Configuration | Yntraa             |
| Database Availability and Monitoring    | Yntraa             |
| Application Schema and Data             | Customer           |
| Application Access and Queries          | Customer           |
