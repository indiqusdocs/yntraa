# Managed Databases Offering 
In this section, the following Databases are provided:
## SutraDB (Relational Databases)
Yntraa SutraDB is the relational database service family under Yntraa Cloud MDBaaS, built to support transaction-intensive and structured data workloads.

SutraDB provides managed deployments of industry-standard relational databases with features such as:
- ACID compliance
- Schema-based data modeling
- Strong consistency and transactional integrity

Common use cases include:
- Web and Application
- ERP and CRM systems
- Financial and accounting applications
- Order management and billing platforms
- Traditional line-of-business applications

Databases offered under SutraDB include MySQL, PostgreSQL, MariaDB, and MSSQL.

## FlexiDB (Non-Relational Databases)
FlexiDB is the non-relational and distributed database service family under Yntraa Cloud MDBaaS, designed for high-scale, low-latency, and flexible data models.

FlexiDB supports databases optimized for:
- Semi-structured and unstructured data
- High-throughput and real-time workloads
- Horizontal scalability and distributed architectures

Common use cases include:
- User profile and session management
- Caching and real-time analytics
- Search, logging, and observability platforms
- IoT, event-driven, and big-data workloads

Databases offered under FlexiDB include MongoDB, Redis, Cassandra, OpenSearch, ElasticSearch, Hadoop, ScyllaDB, and Couchbase.
