# Feature List and Integration Details

Yntraa Storage is delivered as an integration with a leading object storage platform. The integration functions using API-driven checkpoints.

## Standard Integration Touchpoints

- **Account Creation** - When end-users opt for the Yntraa Storage, an account is created on object storage platform. This account allows user to manage and configure their object storage platform deployment, access administrative features, and control various aspects of their storage environment.
- **Cataloguing** - This service is part of the catalogue, i.e., the service provider configures from where to deliver the service and what price to deliver the service.
- **Usage Reporting -** All usage data is available through the UI. Users can log in to Yntraa Cloud at any time to view consumed resources and the associated usage details.

## Specific Integration Touchpoints

- **Create Buckets** - Users can create buckets with unique names, following the naming convention mentioned on Yntraa Cloud.
- **Add Objects** -  Users can add objects to their respective buckets using the Yntraa Cloud.
- **Manage Service on Object Storage Platform** -  Detailed object and bucket management is done via the object storage platform admin panel, which can be accessed directly from Yntraa Cloud.

## Feature List

The following features are available as part of Yntraa Storage, with ongoing enhancements.

- **Single S3 Zone**- Currently, on Yntraa Cloud, only single S3 zone support is available
- **Buckets View** - On the Yntraa Cloud UI, the end user can log in and view all the distinct buckets they have created.
- **Bucket Creation** - User can create buckets with unique names following the naming convention which is mentioned on the Yntraa Cloud UI.
- **Bucket Edit** - End user can edit the already created Buckets at any point in time by logging in to the Yntraa Cloud UI
- **Object Creation**- Users from the Yntraa Cloud UI itself can upload objects.
- **Object Deletion**- Users can delete the objects associated with the bucket straightforwardly and hassle-free.
- **Object Access**- There is a Manage service button on the UI; to open a new tab, click on it with Object Storage Platform URL, and users can log in with their username and password, and simply they can access, manage and configure things related to their storage environment.
- **S3 API Compatibility** - Yntraa Object Storage Service is fully compatible with S3 API.
- **Option to Unsubscribe from the Service** - Users can easily unsubscribe from the service at any time from the Yntraa Cloud UI only.
