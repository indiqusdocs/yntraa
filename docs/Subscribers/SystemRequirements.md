# System Requirements

To ensure optimal performance and full functionality of the Yntraa Cloud Console, users must meet the following system requirements:

## Supported Browsers

The Yntraa Cloud Console is a web-based application and supports the following desktop browsers:

- Google Chrome (latest stable version recommended)
- Mozilla Firefox (latest stable version recommended)
  
## Browser Settings

- JavaScript must be enabled
- Cookies must be enabled for authentication and session handling
- Pop-ups should be allowed for the Yntraa domain to ensure proper functionality

## Device Compatibility

The Yntraa Cloud Console is designed for desktop and laptop use only and is not compatible with mobile devices or tablets.

## Internet Connectivity

A stable and reliable internet connection is required for smooth and uninterrupted use of the console.
