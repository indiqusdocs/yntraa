# Address
1. From the left sidebar, navigate to **Contacts** under PROFILE. The following screen appears with the existing contact details:
![contactsnew](img/contactsnew.png)
- Name
- Email
- Mobile
- Type
- Status
- Portal Access
- Portal Role
- Portal User Status
- Address Status
![contactsnew2](img/contactsnew2.png)
2. Click the **New Address** button. The following screen appears to enter the following details: 
- Name
- Email
- Country Code
- Mobile Number
- Type 
3. Click the **Submit** button to save the new contact.
4. Use the **Search Bar** to find specific contacts.
5. Click on **Choose Columns** to customize the fields displayed in the table.
6. Click on **Edit** icon to modify the details of that specific entry.
