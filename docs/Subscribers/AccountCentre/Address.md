# Address
1. From the left sidebar, navigate to **Address** under PROFILE. The following screen appears with the existing company address details:
![address](img/address.png)
- Address
- City
- State
- Country
- Pin Code
- Type
- GSTIN
- SEZ/Foreign
- Address Status
![address2](img/address2.png)
2. Click the **New Address** button to enter the following details:
- Address
- Country
- State
- City
- Pin Code
3. Select the **GST Applicable** option and enter the GSTIN number in the **GSTIN** field. 
4. Click the **Submit** button to save the new address.
5. Use the **Search Bar** to find specific address.
6. Click on **Download icon** to download the address list.
7. Click on **Choose Columns** to customize the fields displayed in the table.
8. Click on **Edit** icon to modify the details of that specific entry.
9. Click on **Upload Document** icon to attach supporting documents related to the address.
