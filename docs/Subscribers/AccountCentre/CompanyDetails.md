# Company Profile

1. From the left sidebar, navigate to **Company Details** under PROFILE. The following screen appears with the details:
   ![companydetails](img/companydetails.png)
2. Click the **Update** button to update any necessary fields.
   ![companydetails2](img/companydetails2.png)
3. Click the **Save** button to save the changes.
