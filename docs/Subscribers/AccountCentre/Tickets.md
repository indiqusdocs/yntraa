# Tickets

## Ticket Dashboard

 The **Tickets Dashboard** provides a complete summary of ticket activities, including their status, severity, categories. It helps in monitoring and analyzing support tickets effectively.

1. Navigate to One Yotta **Sign In** page. The following screen appears: 
![signinoneyotta](img/signinoneyotta.png)
2. Enter your **Email ID** and **Password** in the required fields.
3. Click on the **Sign In** button to access the dashboard. The following screen appears:
![oneyottadashboard](img/oneyottadashboard.png)
4. From the dashboard, navigate to the **Ticket** tab to view and manage tickets. The following screen appears with details:
![ticketdashboard](img/ticketdashboard.png)
 - **Total:** Displays the total number of tickets raised.
- **Open:** Shows the number of currently open tickets.
- **Closed:** Indicates the number of tickets that have been resolved and closed.
- **In Progress:** Represents tickets that are currently being worked on.
- **Others:** Includes tickets that fall under miscellaneous or undefined statuses.
- **Severity wise Tickets:** Displays tickets categorized based on severity levels (such as Critical, High, Medium, Low) to help prioritize issues.
- **Status wise Tickets:** Shows the distribution of tickets according to their current status (Open, Closed, In Progress, etc.).
- **Assigned Team wise Tickets:** Provides insights into how tickets are distributed among different teams responsible for handling them.
- **Month-wise Resolved Tickets:** Displays the number of tickets resolved over different months, helping track performance trends.
- **Category wise Tickets:** Shows tickets grouped based on categories, such as technical issues, billing, or general queries.
- **Severity vs Team Tickets:** Allows comparison between ticket severity and the teams handling them. A month filter is available to refine the data.
- **Teams S1 - Severity Tickets:** Displays severity-level tickets specifically for the S1 team, with an option to filter data by month.
  
## Creating Ticket
  
Creating a ticket helps users report issues or request services easily. By entering basic details and submitting the form, the request is logged and assigned to the appropriate team for further action.
  
  ![ticketcreation](img/ticketcreation.png)
  
1. Navigate to the **Ticket** section from the left sidebar and click on **New Ticket**.
2. On the **New Ticket** page, select the ticket type:
    - **Incident** 
    - **Service Request**
3. In the **On behalf of** field, select the appropriate account for which the ticket is being raised.
4. Choose the **Project** from the dropdown list (for example, Default Project).
5. Select the **Domain** (for example, OneYotta).
6. Choose the relevant **Category** (for example, Sales).
7. Select the appropriate **Sub-category** (for example, Other).
8. Assign the ticket to the required user/team (if applicable).
9. Enter a clear and concise **Subject** for the ticket.
10. In the description box, provide detailed information about the issue or request.
11. Add any supporting files under **Attachment**
12. After filling in all required details, click on the **Submit Ticket** button.

## Open Tickets

The **Open** tickets section lets you view and manage all active tickets in a single place
 
1. From the left-side menu panel, click on **Open** under the **Ticket** section. The following screen appears:
   ![openticket](img/openticket.png)
2. The open tickets are listed in a tabular format with the following details:
    - **Ticket No** – Unique ticket identifier
    - **Subject** – Brief description of the issue/request
    - **Category** – Type of request (for example, Sales)
    - **Sub Category** – More specific classification (for example, Other)
    - **Severity** – Priority level (for example, Low)
    - **Created On** – Date when the ticket was raised
    - **Status** – Current status (for example, Open)
3. Use the toggle at the top to switch between **Incident** and **Service Request** views.
4. Use the **Search bar** on the top right to quickly find a specific ticket by entering keywords or ticket number.
5. Click on the **Filter icon** to refine results based on specific criteria and use **Choose Columns** to customise which fields are displayed in the table.
6. Click on the **Excel icon** to download the list of open tickets.
7. Click on the **eye icon** to see complete details.
