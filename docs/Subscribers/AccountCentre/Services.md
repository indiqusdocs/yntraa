# Services

1. Navigate to One Yotta **Sign In** page. The following screen appears: 
![signinoneyotta](img/signinoneyotta.png)
2. Enter your **Email ID** and **Password** in the required fields.
3. Click on the **Sign In** button to access the dashboard. The following screen appears:
![oneyottadashboard](img/oneyottadashboard.png)
4. From the dashboard, click on the **Services** tab from the left-side menu. The **Orders** are listed in a tabular format with the following details:
   ![orders](img/orders.png)
   - Order No.
   - Order Period
   - Start Date
   - Start Date
   - Advance Payment Term
   - Total Amount
   - Contract Currency
   - Status
 5. Use the **Search bar** to quickly find specific orders.
 6. Click on **Choose Columns** to customize the fields displayed in the table.
 7. Use the **Excel icon** to download the order list.
    
## Active Services
![allserviceactive](img/allserviceactive.png)

1. From the dashboard, click on the **Assets** tab from the left-side menu.
2. Under the **All SERVICES** section, navigate to **Active**, then either select the **Select All** products option or select individual products under Orders: Active to view the list of active services displayed in a tabular format with details:
- Order No.
- Order Line No.
- Product Family
- Product Name
- Bill Start Date
- Bill End Date
- Bill Currency
- Location
- Status
 3. Use the **Search bar** to quickly find specific active services.
 4. Click on **Choose Columns** to customize the fields displayed in the table.
 5. Use the **Excel icon** to download the active services list.
