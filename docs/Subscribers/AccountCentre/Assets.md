# Assets

1. Navigate to One Yotta **Sign In** page. The following screen appears: 
![signinoneyotta](img/signinoneyotta.png)
2. Enter your **Email ID** and **Password** in the required fields.
3. Click on the **Sign In** button to access the dashboard. The following screen appears:
![oneyottadashboard](img/oneyottadashboard.png)
4. From the dashboard, click on the **Assets** tab from the left-side menu. The asset are listed in a tabular format with the following details:
   ![assets](img/assets.png)
   - Assert Name
   - Assert ID
   - Type
   - Subtype
 5. Use the **Search bar** to quickly find specific invoices.
 6. Click on **Choose Columns** to customize the fields displayed in the table.
 7. Use the **Excel icon** to download the invoice list.
