# Monitoring Infra

## Dashboard

1. Navigate to One Yotta **Sign In** page. The following screen appears: 
![signinoneyotta](img/signinoneyotta.png)
2. Enter your **Email ID** and **Password** in the required fields.
3. Click on the **Sign In** button to access the dashboard. The following screen appears:
![oneyottadashboard](img/oneyottadashboard.png)
4. From the dashboard, navigate to the **Monitoring & Usage**. The following screen appears with the infra monitoring **Dashboard** details:
   ![infradashboard](img/infradashboard.png)
5. The top section displays the status of Server, Network, Firewall, Storage, and URL, showing their uptime and downtime counts.
6. The **Alert Count** section displays alerts for Server, Network, Firewall, and Storage. If no alerts are present, it shows “No alerts found.”
7. The **Uptime** section provides information about system availability and performance.
8. The **Top 5 Utilizations** panel lists CPU Utilization and Memory Utilization details. It includes columns such as Host IP, Host/Visible Name, CPU Utilization, Memory Utilization, and Host Infra.
- Use this dashboard to monitor infrastructure health, track alerts, and analyze resource utilization in real time.
