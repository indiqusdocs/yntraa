# Monitoring Infra

1. Navigate to One Yotta **Sign In** page. The following screen appears: 
![signinoneyotta](img/signinoneyotta.png)
2. Enter your **Email ID** and **Password** in the required fields.
3. Click on the **Sign In** button to access the dashboard. The following screen appears:
![oneyottadashboard](img/oneyottadashboard.png)
4. From the dashboard, navigate to the **Monitoring & Usage**. The following screen appears:
   ![infradashboard](img/infradashboard.png)
5. At the top, the **Alert Count** section displays alerts for Server, Network, Firewall, and Storage. If no alerts are present, it shows “No alerts found.”
6. Below this, the **Uptime** section provides information about system availability and performance.
7. The **Top 5 Utilizations** panel lists CPU Utilization and Memory Utilization details. It includes columns such as Host IP, Host/Visible Name, CPU Utilization, Memory Utilization, and Host Infra Type. If no data is available, it displays “No Data Available.”
    
- Use this dashboard to monitor infrastructure health, track alerts, and analyze resource utilization in real time.
