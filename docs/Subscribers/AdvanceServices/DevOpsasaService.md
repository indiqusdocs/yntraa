# DevOps as a Service

To activate the desired Devop service, perform the following steps:
1. Navigate to **ADVANCED SERVICES** > **DevOps as a Service**. ![dev](img/dev.png)
2. Click the **ACTIVATE** button. ![devactivation](img/devactivation.png)
3. Select the I have read and agreed to the **End User License Agreement** and **Privacy Policy** option, and click **CONFIRM ACTIVATION** button.
