# About One Yotta Account Centre

The **Account Centre** on the Yntraa cloud console is a non-technical (commercial and support) dashboard used for viewing and managing all non-technical aspects of your Yntraa cloud account. You can access the Account Centre by clicking on any item in the user menu at the top-right, the help/support icon in the header, or the wallet icon in the header.

The Yntraa cloud console provides a one-click view switcher between the **Account Centre** and the **Cloud Dashboard**.
